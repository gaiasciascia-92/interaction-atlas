# Interaction Atlas

Un atlante di interazioni per portfolio contemporanei — PWA editoriale per catalogare, studiare, confrontare e sperimentare interazioni, motion, effetti visivi e componenti.

**Tutta la documentazione di progetto è in [`docs/`](docs/README.md).** Partire da lì: nessuna decisione di design vive fuori dai documenti.

Stato: **V1 — Milestone 1** (fondazioni). Vedi [`docs/ROADMAP.md`](docs/ROADMAP.md).

```bash
npm install
npm run dev        # sviluppo (include /styleguide)
npm run build      # produzione (esclude /styleguide)
npm run check      # astro check + typescript strict
npm run lint
```
